# プライバシーポリシー

各アプリのプライバシーポリシーをご確認いただけます。

人生の残り時間を視覚化するアプリです。お客様の個人情報はデバイス内にのみ保存され、外部に送信されることは一切ありません。

[プライバシーポリシーを見る](simple-vocabulary-book.md)
